# Game Tile Package

This folder is reserved for the StakeEngine game tile deliverables. Do not treat existing splash or gameplay screenshots as final tile assets; the tile must be exported against the official StakeEngine tile template/guidelines.

## Required Final Assets

- `background.png`: bright background image that does not clash with the Stake background and avoids dark edges.
- `foreground.png`: transparent foreground focus image with the key subject correctly filling the focus area.
- `title.png`: title treatment placed within the inner guidelines, not too close to any edge.
- `provider-logo.png`: provider logo if required by the submission template.
- `gradient.png`: gradient similar in color to the background.
- `preview.png`: assembled preview for internal review before upload.

## Checklist

- The tile is generally bright.
- The background image is appropriate for the game and does not have dark edge clashes.
- The foreground image is appropriate and fills the key focus area.
- The gradient color family matches the background.
- The title fits inside the inner guidelines.
- No wording or multipliers appear on the background or foreground art.
- No offensive, discriminatory, or inappropriate content is visible.
- Final exported assets match the official StakeEngine tile template dimensions and file requirements.
